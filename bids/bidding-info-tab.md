# Bidding Info Tab on Client Page

The **Bidding Info** tab on the Client page provides a complete view of a client's bidding activity and related settings for the current sale.

## Bidder Numbers

- View and add bidder numbers for the current client
- Displays total spent and maximum credit/bid allowed per bidder number

![Bidder numbers](https://circuitdemo.s3.eu-central-1.amazonaws.com/help-text-images/bidder-numbers.png)

## Credit Requests and Call Requests

Click **Show client credit requests and call requests** to reveal these blocks.

![Show requests toggle](https://circuitdemo.s3.eu-central-1.amazonaws.com/help-text-images/show-requests.png)

### Credit Requests

- Clients can send credit requests for staff approval
- The credit amount determines the maximum the client can win

### Call Requests

- Enter a range or specific items for a client's call request list
- Staff can add notes to call requests
- Print call requests from the call request page

![Credit and call requests](https://circuitdemo.s3.eu-central-1.amazonaws.com/help-text-images/requests.png)

## Enter Bids and Delete All Client Bids

- Enter bids for only the current client (using the bidder number dropdown)
- Option to delete **all** client bids in the sale (requires confirmation)

![Enter and delete bids](https://circuitdemo.s3.eu-central-1.amazonaws.com/help-text-images/enter-delete-bids.png)

## Items Bids Table

- Shows the current client's bids, even when they are not the highest bidder
- Green background highlights rows where the client is the highest bidder
- Popup page navigation is limited to items where the client has placed bids

![Bidder items table](https://circuitdemo.s3.eu-central-1.amazonaws.com/help-text-images/bidder-items-table.png)
