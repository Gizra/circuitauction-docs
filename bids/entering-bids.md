# Entering Bids

Staff can enter bids on behalf of bidders from the Bids page.

## Steps

1. Select the bidder number (bidder name and info will appear below).
2. Enter the lot number (the items table will filter to that lot).
3. Enter the bid amount.
4. Click **Extension** if the bidder requires the item to be expertized.
5. Submit the bid.
6. A confirmation message will appear, and the items table will reload with the current bid calculated.

![Enter bids form](https://circuitdemo.s3.eu-central-1.amazonaws.com/help-text-images/enter-bids.png)

{% hint style="info" %}
You can also enter bids for a specific client from the **Bidding Info** tab on the Client page. See [Bidding Info Tab](bidding-info-tab.md).
{% endhint %}
