# Bids Item Table

The items table displays only items that have received bids, highlighting the top bidder's information along with other important details for each item and its associated bids.

## Filters

Staff can filter the items table by:

- Approval status (approved/unapproved bid status)
- Unknown bidder numbers
- Item status (Open, Sold, Unsold, etc.)

![Table filter options](https://circuitdemo.s3.eu-central-1.amazonaws.com/help-text-images/table-filter.png)

## Table Columns

- **Plus sign** — Opens the [Item All Bids Popup](item-all-bids-popup.md) for that item
- **Lot number** — Linked to the item page
- **Status indicators:**
  - Yellow square — Top bid is unapproved
  - Red exclamation triangle — Item has a private message or is under extension
  - Green checkmark — Sold item
  - Purple group icon — Part of a group bid (hover to see group name and alternative items)
- **Starting price** — Item's starting price
- **Current price** — Calculated current price
- **Bid type** — Mail, floor, agent, etc.
- **Bid amount** — The bid value
- **Bidder name** — Linked to the client page
- **Bidder number**
  - Bid note (yellow exclamation circle) — Indicates a note is attached to the bid
- **Create date** — When the bid was placed

{% hint style="warning" %}
A red background with strike-through text indicates that all bids for the item have been deleted.
{% endhint %}
