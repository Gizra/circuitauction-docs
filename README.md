# CircuitAuction Backoffice Documentation

Welcome to the CircuitAuction Backoffice documentation. This guide will help you understand and use all features of the auction management system.

## What is CircuitAuction?

CircuitAuction is a comprehensive auction management system that handles the complete lifecycle of auctions - from consignment intake and cataloging to live bidding and post-sale accounting.

## Main Features

### Client Management
Manage all your clients - both buyers (bidders) and sellers (consignors). Track their status, approve bidders, assign bidder numbers, and manage their participation in auctions.

### Item Management
Create and manage auction lots. Import items in bulk, assign lot numbers, upload images, set estimates and reserves, and track item status through the auction lifecycle.

### Consignment Tracking
Track items from consignors, manage commission structures, generate consignment statements, and handle post-sale accounting.

### Live Auction
Run live auctions with the clerk screen. Place bids from the floor, phone, and online. Track real-time bidding activity and manage auction sessions.

### Sales Management
Create sales, assign items, set up catalogs, and manage the complete sale workflow from pre-sale preparation to post-sale invoicing.
