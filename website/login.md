# Login

**Selector:** `#login` &nbsp;•&nbsp; *one per page*

## What it does

A sign‑in form (email + password). On success it signs the user in across all blocks on the page and redirects per the URL parameter, or to the home page. Place it on your dedicated login page.

## How to add it

```html
<div id="login" class="circuit-user-ui"></div>
```

## Options

None.

> Interface language follows the `language` attribute on the [`#circuit-setup`](website/README.md#the-setup-div-circuit-setup) div (or the visitor's browser language).
