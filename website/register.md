# Register

**Selector:** `#register` &nbsp;•&nbsp; *one per page*

## What it does

A new‑account registration form: email, password (with confirmation), name and phone, plus reCAPTCHA verification. If your site is linked to partner back offices, an optional consent checkbox to share data with them is shown. On success the bidder is registered and the [My Account](website/my-account.md) view is displayed. If the visitor is already logged in, the block shows My Account instead of the form.

## How to add it

```html
<div id="register" class="circuit-user-ui"></div>
```

## Options

None.
