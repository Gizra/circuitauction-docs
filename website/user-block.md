# User Menu

**Selector:** `#user-block` &nbsp;•&nbsp; *one per page*

## What it does

The account widget for your site header. When the visitor is **logged out** it shows sign‑in / register links. When they are **logged in** it shows their name with an avatar initial and a dropdown menu: *My Account, My Bids & Credit, Billing, My Favorites, My Interests, Logout*. The Billing entry only appears when billing is enabled for your site.

It updates itself automatically when the user logs in or out (no page reload needed).

## How to add it

```html
<div id="user-block" class="circuit-user-ui"></div>
```

## Options

None. The menu renders based on the signed‑in user and your site configuration.
