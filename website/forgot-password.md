# Forgot Password

**Selector:** `#forgotpassword` &nbsp;•&nbsp; *one per page*

## What it does

A password‑reset request form. The bidder enters their email, passes reCAPTCHA verification, and a reset email is sent. A success or error message is shown in place.

## How to add it

```html
<div id="forgotpassword" class="circuit-user-ui"></div>
```

## Options

None.
