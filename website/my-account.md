# My Account

**Selector:** `#my-account` &nbsp;•&nbsp; *one per page* &nbsp;•&nbsp; **login required**

## What it does

The bidder's account‑management page, organized into three tabs:

- **Personal Info** — name, email, phone, address and preferences.
- **Change Password** — set a new password.
- **Addresses** — add, edit, delete and choose a default among multiple saved addresses.

If your site supports more than one language, a language selector is also offered. The block requires the visitor to be signed in; if not, the sign‑in flow is shown.

## How to add it

```html
<div id="my-account" class="circuit-user-ui"></div>
```
