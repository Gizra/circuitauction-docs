# Magazine Subscription

**Selector:** `#subscription-magazine` &nbsp;•&nbsp; *one per page*

## What it does

Shows a library of magazine flipbooks as a grid of covers. Access is gated by the bidder's subscription:

- **Not logged in** — a sign‑in banner.
- **No / expired subscription** — a status message with a link to purchase or renew.
- **Active subscription** — the flipbooks open for reading.

## How to add it

```html
<div id="subscription-magazine" class="circuit-user-ui"></div>
```

## Options

None.
