# My Interests

**Selector:** `#my-interests` &nbsp;•&nbsp; *one per page* &nbsp;•&nbsp; **login required**

## What it does

Lets the bidder manage the auction categories they're interested in, so the system can tailor notifications and recommendations:

- **Add categories** — autocomplete search to find and add new categories.
- **My categories** — a table of auto‑detected and manually added categories, with a hierarchy search, an active/inactive filter, sortable columns (name, hierarchy, bids, won, favorites) and an on/off toggle per category. Collapses to cards on small screens.
- **Additional interests** — a free‑text field for interests that aren't tied to a category.
